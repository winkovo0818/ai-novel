# 第 1 章《雨夜火房》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：2177

雨砸在柴饦峰火房的破瓦上，声音稠得像煮烂的粥。

沈言蹲在灶膛前，湿柴塞进去，浓烟立刻倒呛回来，呛得他眼眶发涩。他没起身，也没躲。三年了，他学会的第一件事就是别在柴饦峰表现得太灵光。

火房执事周奉站在门口，油布伞沿滴下水线，刚好砸在沈言脚边的泥坑里。

“后山裂井的柴，还没劈完。”

不是问句。沈言低头，手指抠着灶膛边缘的炭灰：“雨太大，斧头钝了。”

“钝了就磨。”周奉的声音没什么起伏，“明天门内小比，你如果还是垫底，柴饦峰不留废物。”

沈言盯着自己指甲缝里的黑灰。三年不出师即逐出，这是铁律。他去年勉强挨过，今年考核再垫底，连柴饦峰也待不下去。废柴宗门收留弃徒，但弃徒里的废物，也没人要。

他应了一声：“知道了，周执事。”

周奉没走。伞下那双眼睛扫过沈言佝偻的背，停了几息，才转身离开。雨声吞没脚步声。

沈言继续往灶膛里添湿柴。浓烟裹着木头腐烂的酸气涌出来。他得忍。现在还不是时候。剑魂三天前才刚有苏醒的迹象，像埋在冻土下的种子，拱得他脊椎发麻。他不能让任何人看出来。

后山裂井。那地方他去过。废柴宗门说是上古遗留的剑冢，其实早就荒了，井口长满青苔，扔块石头下去，半晌才听见闷响。周奉让他去劈那里的柴，根本就是刁难。裂井旁边的树全是死木，硬得像铁，斧头砍上去只崩刃。

但他必须去。柴饦峰的柴火堆是宗门最不起眼的地方，却挨着后山的禁制边缘。上个月他去送柴，经过裂井时，脊椎里的东西猛地烫了一下。就一下。

他得搞清楚那是什么。

湿柴终于燃起一点火苗，舔着锅底。沈言站起身，膝盖关节发出轻微的咔哒声。火房角落堆着今天要劈的柴，死木颜色发黑，沉得压手。他挑了最粗的一根，扛上肩。木头硌着肩胛骨，疼得实在。

雨没停。他推开火房后门，雨水立刻糊了满眼。柴饦峰的后山小路泥泞，深一脚浅一脚，靴子很快灌满泥水。裂井在半山腰一片塌了半边的旧墙后，井口果然长满湿滑的青苔。他把死木靠在墙根，拿起插在泥里的斧头。斧刃果然卷了好几处。

他抡起斧头，砍下去。

铛的一声，震得虎口发麻。死木上只留下一道白印。

再砍。还是白印。

雨顺着脖颈灌进衣领。他停下喘气，抹了把脸。不能急。剑魂认主不可逆，如果那东西真的和他有关，总会再有动静。

他换了个角度，斧刃对准木纹的缝隙。这次用巧劲。铛！手腕酸得发颤，但那道缝隙像是宽了一点。

就在这时，脊椎深处猛地一跳。

不是疼，是某种震动，顺着骨头向上爬，直窜到后脑勺。沈言手一松，斧头砸进泥里。他僵在原地，耳朵里全是雨声，可那雨声底下，好像还有别的东西。

很轻，像风吹过生锈的铁管。

他转头，看向裂井。

井口黑黢黢的，什么也看不见。但脊椎里的震动越来越明显，带着一种……饥渴。它在叫。

沈言的手指抠进墙缝的青苔里。冰凉。他该走。现在转身回火房，当什么都没发生，明天照常去考核，垫底，被逐出宗门。这是最安全的路。

可他没动。

父亲失踪前，最后去的地方就是后山裂井。宗门说是失足坠井，尸骨无存。但沈言记得，父亲走前那晚，背着他磨了半夜的剑，说：“言儿，剑魂认主，就是拿命拴在一起。别信那些宗门长老的话，他们找的不是剑魂，是奴役剑魂的法子。”

父亲也是剑修。一个被废柴宗门捡回来，却始终没能觉醒剑魂的剑修。

沈言松开抠墙的手。泥从指甲缝里掉下来。

他走向裂井。

井壁湿滑，长满暗绿色的苔藓。他趴在井沿，往下看。黑。深不见底的黑。雨点落进去，连回声都没有。

可脊椎里的东西在催他。

他吸了口气，对着井口，低声问：“谁在？”

雨声太大，他自己都听不清。

他又问了一遍，声音压得更低：“你是……剑魂？”

脊椎的震动骤然加剧，变成一种尖锐的嗡鸣，刺得他眼前发黑。紧接着，一个声音直接在他脑子里炸开：

。沈昊之子。

四个字。不是疑问，是确认。

沈言浑身的血都凉了。沈昊，是他父亲的名字。

那声音又来了，带着铁锈摩擦般的沙哑：

。终于等到你了。

他猛地向后跌坐在泥水里。后背撞上墙根的死木，疼得他倒抽冷气。脊椎的震动还在，但平息下去，变成一种温热的余韵，贴着骨头。

他盯着裂井。雨还在下。

谁在等？等他干什么？

父亲的名字。剑魂认识父亲。

沈言爬起来，手抖得厉害。他抓起那根死木，这次没砍，直接扛上肩。死木沉得压弯他的腰，但他一步步往山下走，没回头。

回到火房时，天快黑了。周奉不在。沈言把死木扔在柴堆边，生起火，把湿透的外衣脱下来烤。火苗跳动，映着他苍白的脸。

脊椎里还留着那点温热。

门突然被推开。一个杂役弟子站在门口，雨水从他蓑衣上淌下来：“沈言，门主召见。现在。”

火苗猛地一窜。

沈言抬头，看见杂役弟子脸上没什么表情。柴饦峰的杂役，向来不多话。

他抓起半干的外衣套上，跟着走出去。雨小了些，但天更黑了。通往宗门大殿的路铺着青石板，积水映着惨白的天光。

杂役弟子提着灯笼走在前面，光晕在水洼里晃。

沈言摸了摸自己的后颈。那里皮肤微温，像被谁的手指按着。

剑魂认主不可逆。那么，它认的是沈昊，还是沈言？如果认的是父亲，父亲为什么失踪？如果认的是他，为什么是现在？

大殿的灯火在雨幕后透出来，昏黄一片。

杂役弟子停下脚步，回头看他：“门主在等。考核的事，今天就要定。”

沈言看着大殿敞开的门，里面黑沉沉的，看不见人。

他抬脚，踩上湿滑的石阶。
