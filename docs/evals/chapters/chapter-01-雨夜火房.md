# 第 1 章《雨夜火房》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：3135

柴饦峰的火房终年不见光。
沈言蹲在灶膛前，往里头塞湿柴。浓烟涌出来，呛得他喉咙发紧，眼泪混着额头的汗一道往下淌。雨水从漏瓦滴落，砸在灶台边沿，溅起灰黑色的泥点。他用袖子抹了把脸，袖口磨得发硬，蹭得皮肤生疼。

门外脚步声踩着水响。
火房的木门被一脚踹开，潮气卷着风扑进来。执事孙奉站在门口，蓑衣往下滴水，手里那截戒尺黑得发亮。他扫了一眼灶膛里半燃不燃的柴火，没说话。

沈言站起来，膝盖发麻，垂手退到墙角。墙角堆着去年剩下的枯枝，霉味钻进鼻子。他低头看自己鞋面。左脚那只破了个洞，露出的脚趾沾了灶灰。

“明日辰时，门主要见你。”孙奉的声音像生锈的铁片刮过石板。
沈言肩膀缩了一下。这是真怕。门主三年来只召见过两次人，上次那个外门弟子被叫去后，再没回过柴饦峰。

“听见了？”
“听见了。”沈言声音压得很低，像怕惊动梁上结网的蜘蛛。

孙奉往里走两步，戒尺敲了敲灶台边缘。灰烬簌簌往下落。“柴火潮成这样，你是想让明日早课的弟子都吃夹生饭？”
沈言不接话。他知道不是柴的问题。上月他撞见孙奉私吞外门供给的灵炭，从那天起，罚劈的柴火就总也劈不完，罚烧的灶火总也烧不旺。忍下来，是因为他还没到出师年限，现在走，三年苦役白费。

孙奉盯着他看了半晌，忽然笑了一声。那笑声短促，像呛了风。“后山裂井的锁链松了。你去一趟，把铁链重新绞上。”
沈言抬起头。后山裂井是禁地，传闻镇着东西，但谁也没说清镇的是什么。三年前有弟子靠近过，回来后高烧三天，醒来什么都不记得。

“现在？”雨声更大了，敲得屋顶砰砰响。
孙奉把蓑衣脱下来，扔到他脚边。“子时前回来。误了明日早课，你知道后果。”

沈言捡起蓑衣。蓑草扎手，带着孙奉身上的烟熏味。他穿上，草绳在颈前打了个死结。出门时孙奉已经不在了，只有那截戒尺横在门槛上，像一道矮矮的栏杆。

后山的路泥泞。
蓑衣挡不住多少雨，沈言走到半途，裤腿就湿透了，贴在小腿上，每走一步都往下坠。林子里黑，只有偶尔闪电劈下来时，能看见前方石阶的轮廓。石阶缝隙里长满青苔，踩上去滑得厉害。他摔了一跤，手掌撑在碎石上，划开一道口子。血珠渗出来，很快被雨水冲淡。

裂井在山坳深处。
远远就能看见那口石砌的圆井，井口横着四道碗口粗的铁链，其中一道确实松了，垂在地上，另一头没入井中。井沿刻着模糊的符文，雨水积在凹槽里，映出远处柴饦峰上火房那点昏黄的光。

沈言走过去，弯腰去捞铁链。
铁链冰得刺骨，像握住一截死人的骨头。他使劲往上提，链环卡在井沿石缝里，纹丝不动。他换了个姿势，双手都抓上去，脚蹬在井沿外侧借力。铁链动了一下，发出沉闷的刮擦声。

就在这时，井底传来声音。
不是水声，也不是链环碰撞。是一个声音，很轻，像有人隔着厚厚的棉絮说话，但字句异常清晰：“……沈……言……”

沈言手一松，铁链砸回地上。
他往后退了半步，脚后跟磕到石块，踉跄着坐倒在泥水里。心脏跳得厉害，撞得胸腔发疼。井口黑黢黢的，雨滴落进去，连回声都没有。

那声音又响了。
这次更清楚些，带着某种生涩的震动，像井底有什么东西正在撕开自己的喉咙：“你……终于……来了……”

沈言撑着地站起来。手掌沾满泥浆，混着之前伤口的血，黏糊糊的。他往前挪了一步，井口的风往上涌，带着铁锈和另一种味道。很淡，像烧焦的玉，又像很久以前他在父亲书房闻过的墨锭。

父亲。
这个词让他喉咙发紧。十年前父亲失踪，留下的只有一间空书房和这副先天不足的躯体。废柴宗门收留他，给口饭吃，条件是三年苦役换一个外门弟子的名分。他熬了两年零八个月，体内忽然多出一团东西，冰凉的，沉在丹田最深处，像一块拒绝融化的寒铁。他没告诉任何人。

井底的声音沉默了片刻。
然后，一道极细的光从井口缝隙里漏出来。不是火光，是青白色的，像月光被碾碎成粉，涂在铁链上。光沿着链环往上爬，爬到沈言刚才抓握的地方，停住了。那声音再次响起，这次带着某种试探的意味：“你……听得见……为什么……”

沈言没回答。
他在想孙奉。孙奉让他来后山，绝不是为了绞铁链。柴饦峰上下都知道裂井有异，但门主从不提起，孙奉却偏偏在门主召见他的前夜把他支过来。如果他死在这里，或者像三年前那个弟子一样疯掉，明日辰时就不用出现在门主面前。

可如果他不去呢？
孙奉会告他违令。三年苦役白费，他会被赶出宗门。一个体内封着不明之物的弃徒，在九州仙门衰落的世道，活不过三个月。

他抓住铁链。这次没用力，只是让手掌贴上去。青白色的光绕着他的手指转了一圈，冰凉感顺着手腕往里钻，钻到手肘时停住了。那声音说：“你……体内……有剑……”

沈言呼吸停了一瞬。
剑魂。他约略听过这个词。上古遗留的力量核心，能承载它的人万中无一，一旦认主不可逆。父亲书房的书被搬空前，他曾偷翻过一册残卷，上面画着扭曲的符文和一句话：“剑醒之时，旧债新偿。”

他张了张嘴，声音干涩：“你是谁？”
井底沉默了很久。雨声灌进耳朵里，衬得那沉默格外漫长。然后，声音再次响起，这次每个字都像从牙缝里挤出来：“我是……你父亲……留下的……”

闪电劈下来。
白光瞬间照亮整个山坳，沈言看见井底深处有什么东西在动。不是锁链，也不是水。是一团影子，边缘模糊，中间有一道竖起的亮线，像一只闭着的眼睛。亮线睁开了一条缝，青白色的光涌出来，照得井壁符文全部浮现。

光涌到他脚下。
沈言低头，看见自己的影子被拉得很长，一直延伸到井口。影子的头部位置，一点微光正在闪烁，和他胸口的位置重叠。那是丹田里那团寒铁在共鸣。

“明日……门主……会测你的根骨。”声音断断续续，像随时会消散，“剑魂……一旦被他们发现……你活不过当晚……”

沈言蹲下来。
雨水顺着蓑衣边缘滴到井沿上，汇成细流。他伸手按在井口石符上，石头湿滑，但底下有震动，一下一下，和他心跳的频率渐渐重合。“所以孙奉让我来。”他说，“他知道我会死在这里，或者疯掉。这样就不必惊动门主。”

井底那道竖起的亮线颤了颤。
“他……不知道……我在这里……”声音说，“他只知道……这口井……危险……”

沈言站起来。
他抓着铁链，把它拖到井沿上，开始一圈一圈地绞。链环摩擦石槽，发出刺耳的嘎吱声。他没有看井底，只盯着自己的手。手上全是泥和血，但握得很紧。

绞到第七圈时，声音又响了。
“你……不怕……？”
沈言摇头。怕有什么用。怕不会让明日辰时迟来，不会让孙奉消失，不会让他体内这团东西消失。他需要一个答案，关于父亲，关于剑魂，关于为什么是他。

铁链绞紧了。
那道松开的链环重新嵌进石槽，发出沉闷的咔嗒声。井底的光暗下去，那团影子缩回深处，青白色的亮线重新闭合。声音最后说了一句话，很轻，差点被雨声盖过：“明日……测骨时……握住你的右手……掌心……”

然后什么都没有了。
只有雨。沈言站在井边，看着被绞紧的铁链。雨顺着链环往下流，在井沿积成一滩。他松开手，掌心火辣辣的，有几处皮被磨破了。

下山时他没摔跤。
蓑衣重了许多，压在肩上。火房的光还在漏，孙奉不知什么时候回来了，坐在灶膛前烤火，那截戒尺就放在脚边。沈言推门进去，把湿蓑衣脱下来，挂回门后。

孙奉抬眼看他。
“绞上了？”
沈言点头。他走到灶膛前，继续往里塞湿柴。火苗舔上柴皮，发出滋滋的声响，烟雾升腾起来。

孙奉盯着他看了很久。
然后站起身，拿起戒尺，走到门口时停了一下：“明日辰时，门主问你话，照实答就是了。”
门关上，脚步声远去。

沈言往灶膛里又塞了一截柴。
火旺起来，映得他脸上忽明忽暗。他摊开右手手掌，掌心伤口已经不流血了，结着薄薄的痂。痂下有什么在动，很轻，像心跳，又像井底那团影子闭合前最后的颤动。

明日辰时。
他握紧手掌，痂裂开一道细缝，渗出一点血珠。血珠落在灶膛灰烬里，滋一声灭了。
