# 第 5 章《门主旧案》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：2349

沈言坐回木箱边，膝盖抵着箱沿。柴刀横在腿上，他捡起磨刀石，顺着缺口的刃口推过去。沙，沙。声音很轻，被屋外持续的虫鸣盖住大半。

黑牌贴在左胸中衣里，隔着一层粗麻布料。沉渊剑柄带来的紊乱感还在，像一团将散未散的雾，裹着那枚牌。麻意弱了，钝了，但没消失。

他停下磨刀的动作。

不是因为累。是因为怀里的牌在动。

不是之前那种针扎似的麻。是震动。细密的，持续的，从牌心深处传出来，震得他胸骨发闷。沈言左手按在衣襟上，掌心压住那块凸起。震动透过布料，一下下敲进他的皮肤。

试炼在即。辰时将近。

黑牌的震动突然加剧了一瞬，又停了。像有什么东西在牌内挣扎了一下，被按了回去。沈言手指收紧，指节抵着木箱边缘，用力到发白。他闭上眼睛，调动起脊椎深处那点剑魂的感知，小心翼翼地探向怀中的牌。

没有沉渊剑柄那种冰冷的、带着腐朽感的排斥。这次是另一种力道。更规整，更隐蔽，像一张网收紧时的绷劲。

有人在催动黑牌。

沈言睁开眼。窗外的天色又亮了一些，灰白的光透进来，照见他手指缝里磨刀石的粉末。他放下柴刀，手伸进怀里，捏住黑牌的一角，抽出来。

牌面在昏光下泛着哑光。他把它举到眼前，凑近了看。牌面中央，一道极细的裂缝，比头发丝还细，从边缘一直延伸到中心。那是他上次接触沉渊剑柄后留下的。裂缝周围，此刻正浮起一层极淡的、差点看不见的灰色光晕。光晕随着牌身的震动，一明一暗。

禁制在自我修复，也在被外力牵引。

沈言把牌翻过来，看背面。背面光滑，什么都没有。他指尖摸过裂缝的末端，停住了。那里，裂缝分岔，形成一个极小的、箭头似的形状，指向牌的左下角。

左下角。沈言视线移过去。那里原本该是平整的牌面，现在却有一个针尖大的凸起，差点看不出来。他用指甲抠了抠，抠不动。是铸在牌内的。

他想起沉渊的话。黑牌是监控。是眼睛。现在，眼睛在被别人盯着，也在被它自己盯着。裂缝是伤，也是破绽。分岔的箭头，指向的凸起，是什么？

沈言把牌收回去，贴着胸口。震动还在，但规律了，一下，一下，像心跳。他不再尝试压制，任由它震。压制没用，对方要的只是确认“材料”还在。他起身，走到墙角木箱边，蹲下。

箱子里东西不多。几件换洗的旧衣，叠在角落。磨刀石和柴刀拿出来了。剩下最底下，压着一个扁平的油布包。

沈言把油布包抽出来。布包很旧，边角磨损，沾着陈年油渍。他解开系绳，里面是一卷泛黄的麻纸。纸很脆，边缘卷起。他小心地摊开。

纸上是炭笔画的图。线条粗犷，勾勒出一个井口的形状，井壁上标着几处凹陷的位置。旁边有字，字迹潦草：“第四层，左三，深尺二，有硬物。疑为旧制禁制基点。”落款是个“昊”字。沈昊。

沈言手指抚过那个“昊”字。父亲的字迹。他从未见过父亲写字，只在幼时模糊的记忆里，听过这个名。

他继续往下翻。麻纸只有三张。第二张画的是黑牌的草图，正面背面都画了，背面标注了几个点，用炭线连接，像某种阵纹。旁边写着：“牌内禁制根系在此。破其根，网可暂歇。”第三张是文字，字迹更潦草，有些地方被水渍晕开，只能辨认出片段：“……蒋阶以‘匠人’自居，实则……弃徒皆料……魂炼成，则料碎……吾儿……血脉……恐亦难逃……”

麻纸从沈言手里滑落，掉在箱底。

他蹲在原地，没动。手指抠进箱沿木缝，抠得生疼。父母之死。失踪。意外。这些词在他脑子里转，转成了炭笔画的井口，转成了黑牌背面的阵纹，转成了“吾儿……血脉……恐亦难逃”。

不是意外。是献祭。蒋阶需要他们的魂魄去炼剑魂。自己活下来，只是因为成了新的“容器”。

箱底，黑牌还在震动。一下，一下。

沈言捡起麻纸，卷好，塞回油布包。系绳时，手指碰到箱底一块硬物。他拨开旧衣，摸到一个巴掌大的、扁平的陶片。陶片粗糙，边缘不规则，像是从什么大器皿上敲下来的。他把它拿出来，翻过来。

陶片背面，用尖锐的东西刻着字。字很小，深陷进陶土里：“第四层，左三。井壁夹层。牌存此，可暂避。”没有落款。

沈言捏着陶片。陶土冰凉，吸走了他指尖的温度。第四层，左三。和父亲画的图上标记的位置一样。父亲把一块刻了字的陶片藏在箱底，藏在他每天都能摸到的旧衣下面。

他为什么没发现？因为这箱子是周奉三年前扔给他的，说装杂物。父亲的遗物，只有这卷麻纸，这块陶片。还有那把柴刀。

沈言把陶片翻过来，对着窗外透进的灰光。字迹边上，还有一行更小的刻痕，差点被磨平了，要侧着光才能看见：“吾妻阿砚，亦为此间魂。”

母亲。

他盯着那行字，直到眼眶发酸。然后他把陶片和油布包一起，塞进怀里，贴在黑牌旁边。硬邦邦的两块东西，硌着胸口。

屋外，远处传来脚步声，不止一个人。是杂役弟子在巡查各处住处，清点人数。辰时到了。

沈言站起身。他拿起柴刀，刀刃在磨刀石上蹭过，缺口还在，但锋刃亮了一线。他把刀插回腰间，拉了拉粗麻衫的衣襟，把怀里的东西压平整。

黑牌还在震。陶片和麻纸沉甸甸的。

他走到门边，手搭在门闩上。门外，脚步声近了，混着低声的催促。沈言没开门。他转身，走回木箱边，蹲下，把箱底最下面那块松动的砖抠起来。砖下是个浅坑，坑里什么都没有。

他把陶片拿出来，放进坑里，用砖盖好。油布包留在怀里。

父母之死不是意外。是献祭。蒋阶需要他们的魂魄去炼剑魂。自己活下来，只是因为成了新的“容器”。现在，容器要去试炼了。带着父亲留下的线索，和母亲留下的那行字。

沈言重新走到门边。这一次，他拉开了门闩。

门外，灰白的天光涌进来。两个杂役弟子站在院里，见他出来，其中一个皱眉：“磨蹭什么？裂井集合，辰时三刻不到，按缺席论处。”

沈言低头，跨出门槛。湿泥的土腥气扑面而来。他没说话，只跟着那两个弟子往院外走。怀里的黑牌震得更密了，陶片的边角硌着肋骨，硬邦邦的疼。

他摸了摸腰间的柴刀。刀柄被汗渍浸得滑手。
