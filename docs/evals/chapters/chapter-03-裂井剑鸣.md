# 第 3 章《裂井剑鸣》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：2651

山道上的泥比火房里的更深。沈言把裤脚卷到膝盖，每走一步，小腿都得从黏土里拔出来。夜雨没停，却不如柴饦峰上那般劈头盖脸。细密的雨丝打在脸侧，凉意钻进衣领。

怀里揣着那块“丙七”木牌。正面的字被体温焐得有些发烫，背面那道新刻的痕，他用拇指反复摩挲过，很浅，刻得仓促。孙奉没说断碑具体在裂井何处。只说下去，找着，按上。

他记住了木牌第一次嗡鸣的时辰。孙奉刚递过来，他接住的瞬间，掌心像被细针扎了一下，不是疼，是麻。当时灶膛的烟正浓，他借着低头咳的动作，瞥见木牌背面那道刻痕里，有极淡的、灰白色的东西流动了一瞬。像活的。

现在那股麻痒感又从掌心泛起来，比在火房时明显。沈言停下脚步，侧耳。只有雨声，和山下柴饦峰方向约略的犬吠。他把木牌掏出来，就着天上偶尔漏下的、被云层滤得发青的月光看。刻痕是灰白色的，没有异样。但他知道它动过。像里面有什么东西，被这山里的什么东西吸引住了。

裂井在后山背阴处。不是井，是一道地裂，传说是百年前某位前辈一剑劈出来的。劈完人就疯了，跳了下去。宗门拿铁栅栏围了，日久锈蚀，如今只剩下半截歪斜的桩子。沈言拨开比人还高的荒草，一股浓重的土腥气混着更深的、像是铁锈泡在水里的味道涌上来。

他走到裂缝边缘。往下看，只有一片墨色。风从裂缝里往上涌，不凉，却带着湿重的压迫感，吹得他额发往后贴。木牌在掌心猛地一跳。

这次不是麻。是清晰的震动，嗡鸣声直接在他掌心响起来，像握着一只被困住的蜂。背面那道刻痕，灰白色的东西真的在流动，速度不快，却稳定，沿着刻痕的轨迹循环。沈言想起孙奉说的“按到断碑上”。断碑在下面。木牌在共鸣。这里面有联系，不止是孙奉的任务那么简单。

他蹲下身，摸了摸裂缝边缘的石头。石头是冷的，但指尖触碰的瞬间，掌心木牌的震动加剧了。下面有东西在叫它。或者说，在呼应它。

体内深处，那道自他懂事起就盘踞在骨血里的、沉默的“剑魂”，此刻也跟着动了动。很轻微，像沉睡中被人碰了一下手指。沈言喉咙发紧。剑魂和这块木牌，和下面的裂井，有关系。孙奉知不知道？蒋阶知不知道？让他下来，只是为了测试？还是为了……喂食？

他必须下去。不是因为孙奉的命令。是因为答案可能就在下面。他爹娘到底是怎么死的，体内的剑魂到底是什么，这些问题像柴饦峰火房里永远烧不尽的湿柴，闷着，烟一直往他心底里灌。现在，裂缝下面有风往上吹。

沈言解下腰间早就备好的麻绳。一端系在旁边半截石桩上，用力拽了拽。石桩晃动，簌簌掉下碎石。他改了主意，把绳子一端绑死在自己腰上，另一端缠了几圈在手上，然后把木牌塞回怀里。他得留着手。下面有什么在等着，他不知道。

脚探进裂缝。踩到的不是预想的光滑石壁，是凸起的、像是人为凿出的踏脚坑。他一点点往下放身体。黑暗迅速吞没上来，头顶的天光变成一条窄缝，然后连那条缝也被荒草和雨雾遮住。耳边只剩下自己的呼吸，绳子摩擦石头的轻响，还有越来越清晰的、从下方传上来的嗡鸣。

不是木牌了。是下面本身就在响。低沉，持续，像巨大的心脏在缓慢搏动。

落脚坑越来越密。沈言估摸着下了三四丈深，脚下忽然踩空。他身体一坠，腰间绳子瞬间绷直，勒得他眼前发黑。等稳住，才发现是到了一个平台。裂缝在这里拐了个弯，横向扩开，形成了一个略为宽敞的洞窟。空气更潮，铁锈味淡了，多了种更难以形容的、类似陈年血垢的腥气。

他松了绳子，往前摸。指尖触到冰冷的石壁，凹凸不平。掌心的木牌震动得厉害，差点要从他怀里跳出来。沈言把它掏出来。灰白色的流光在刻痕里快速游走，映亮了周围一小片石壁。

石壁上有字。很旧，是用利器直接凿上去的，笔画边缘都崩裂了：

“勿信碑下言。”

四个字，凿得深，凹槽里积着湿气。沈言的瞳孔缩了一下。谁留的？警告谁？

“吱。”

一声尖锐的、像金属摩擦的声响从洞窟深处传来。沈言猛地转身，背靠石壁。木牌的嗡鸣骤然拔高，变成尖啸，同时，怀里的剑魂也剧烈震荡起来，不再是沉睡被碰，而是像遇到了同类，或者天敌。

黑暗中，两点幽绿色的光点亮起，悬浮在半空，一高一低，像眼睛。

“……木牌……”一个声音响起来，干涩，断续，不像从嘴里发出，更像石头摩擦，“你……拿着它……”

沈言屏住呼吸。绿光没有靠近，但压迫感如实质。他把木牌举到身前，让上面灰白色的流光更明显些。“孙奉让我来的。”他说，声音比自己预想的稳，“把这东西，按到断碑上。”

“孙奉……”那声音重复了一遍，带着一种粘稠的咀嚼感，“他……还活着……”

“你是谁？”沈言问。绿光的“眼睛”微微闪动了一下。

“……碑……在下面……更深……”那声音没回答，自顾自地说，“你身上……有……我熟悉的味道……”

沈言心脏重重一跳。熟悉的味道？剑魂？还是别的什么？他想起石壁上“勿信碑下言”的警告，又看向那两点绿光。“我爹叫沈牧，我娘叫柳蕴。”他直接说，盯着绿光，“你认识他们吗？”

绿光猛地跳动了一下。洞窟里的嗡鸣声骤然变化，不再是单纯的低沉，掺进了某种尖锐的、差点要刺破耳膜的频率。那声音变得急促，破碎：

“沈……牧……柳……蕴……名字……他们的名字……不……不能说……说出来……它就会醒……”

“它是什么？”沈言追问。

“……剑魂……不……不止是剑魂……”绿光剧烈明灭，“木牌……是钥匙……也是枷锁……孙奉……想用你……喂饱下面……好让他……拿走该拿的东西……”

该拿的东西？沈言脑子里飞快转着。孙奉，蒋阶，测骨，裂井，剑魂，父母的名字。一条模糊的线串起来，但还差关键一环。“你帮我。”沈言开口，语气是自己没预料到的平静，“你告诉我真相，我帮你出去，或者……做别的。”

绿光静止了。那声音沉默了很久，久到沈言以为它不会回答了，才幽幽传来：

“……契约……老规矩……你问三个……我答三个……然后……你帮我把‘枷锁’……插进碑心……不是按上……是插进去……毁掉它……”

三个问题。沈言舔了舔发干的嘴唇。他有无数个问题。但他只有三个机会。木牌在手里滚烫，剑魂在体内躁动。洞窟深处，像还有更沉重的东西在呼吸。

“第一个问题，”沈言说，“我父母，是被人害死的吗？”

绿光闪动，那声音带着一种奇异的、混合着嘲讽和悲哀的腔调：

“……他们……是自己走进来的……为了封住‘下面’……把剑魂……留给了还在襁褓里的你……然后……死在了……断碑之前……”

沈言站在那里，雨水、汗水、泥浆的味道像是都从记忆深处翻涌上来。他没再问第二个。他走到石壁边，用手背抹了下眼睛，很慢。然后他转身，看向那两点绿光。

“带我去断碑。”
