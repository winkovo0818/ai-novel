# 第 3 章《裂井剑鸣》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：4337

雨停了。

石阶上的积水映着天光，青灰一片。沈言转身往柴饦峰走。湿透的短褐贴在背上，风一吹，凉意钻进骨头缝。他走得不快，靴子踩在湿泥上，发出噗噗的闷响。

回到那间窄屋，他把门掩上。屋里没点油灯，只有窗纸透进来一点昏光。他站在门后，手指摸到怀里那块黑牌。

指尖刚触到牌面，脊椎深处就传来针扎似的麻。不是之前那种震动，是更尖锐的警示。像有细刺顺着骨髓往脑仁里钻。

沈言把手从怀里抽出来。

他走到窗边，借着窗外最后一点天光，摊开手掌。黑牌躺在掌心，表面光滑，边缘有细微的磨损。他调动起脊椎里那点微弱的感知。像伸出看不见的触须，小心地碰了碰牌面。

排斥感立刻涌上来。不是剑魂的愤怒，更像……厌恶。这东西不干净。牌身内部有晦涩的波动，像水底下游动的东西。周奉说“贴身带着”。带着这东西进裂井，等于举着灯笼在暗夜里走。

他没有把牌扔掉。扔了，周奉马上会知道。试炼还没开始，就把底牌亮了。

沈言把黑牌塞回最里层的衣襟，贴着皮肉。那股麻意隔着布料传来，像冰锥抵着皮肤。他解开湿透的外衣，拧了拧水。水滴落在地上，渗进泥地的缝隙。

三年前初到柴饦峰时，他住过更差的地方。柴房角落，和老鼠抢铺盖。那时候周奉还不是执事，只是个管杂役的弟子。看他瘦得脱形，扔了块硬得像石头的杂粮饼过来。

“吃。饿死了没人收尸。”

沈言把饼啃完了。半夜胃里绞着疼，他蜷在草堆里，没出声。

现在周奉成了执事。沈言成了他手底下砍柴的。

窗外彻底暗下来。远处山脊的轮廓模糊成一片黑。沈言吹熄桌上的残烛。那烛火早就灭了，只是习惯性地做个动作。他在黑暗里坐了一会儿，直到眼睛适应了昏暗。

然后他起身，从床底下摸出一把短柴刀。

刀刃钝了，他没去磨。钝刀砍柴费力，但能用。他握着刀，推开后窗。

窗外是柴房的后墙，再往后，就是通往后山裂井的小路。

他翻窗出去。

夜里的山路不好走。昨夜的雨把土泡软了，踩下去没过脚踝。沈言一手抓着柴刀，一手扶着旁边的树干。树皮粗糙，扎手。他走得慢，每一步都先试探地面是否结实。

裂井在后山北坡。

说是井，其实是条裂开的山缝。早年宗门开采灵石留下的矿脉，灵石采尽，矿道坍塌，只留下这条深不见底的裂缝。里面常年阴湿，偶尔能听见底下有水声。柴饦峰的弟子偶尔被罚去裂井附近拾柴，但从不让下去。说是有瘴气，也有说底下封着不干净的东西。

沈言在井口停下。

裂缝宽约三尺，往下看，黑得吞光。井口边缘的石头湿滑，长着暗绿色的苔藓。一股土腥气混着铁锈味飘上来。他蹲下身，摸了摸井壁。石头冰凉，指尖沾上黏腻的湿气。

脊椎里的麻意在靠近井口时忽然变了。

不再是针对黑牌的尖锐排斥，变成一种低沉的嗡鸣。像琴弦被拨动前的那一瞬。剑魂在回应什么。

沈言从怀里摸出火折子。他没点，只是握着。竹筒表面粗糙，划手。他盯着井下的黑暗，呼吸放得很轻。

下面有东西。

不是野兽，也不是寻常的邪祟。剑魂的感知很模糊，只传回“存在”和“古老”两种意味。还有……悲伤？沈言不确定。那情绪太淡，像水底泛起的气泡，还没浮到水面就破了。

他必须下去。裂井是明天试炼的地点。他得在天亮前弄清楚，底下到底有什么。

沈言把柴刀插在腰后，开始往下爬。

井壁湿滑，差点没有落脚的地方。他只能抠着石缝，一点点往下挪。指尖很快磨破了皮，血混着泥水，黏得打滑。他停了几次，呼吸粗重。有一次脚下踩空，整个人往下一坠，肩膀撞在凸出的石棱上，疼得他眼眶发酸。

他没出声。咬紧了牙，继续往下。

越往下，空气越潮。带着一股霉味，像打开多年未动的地窖。光线彻底消失了，他只能凭触觉判断距离。腰间的黑牌贴着皮肉，麻意持续不断，但不再那么尖锐了。或许是离剑魂的源头近了，或许是黑牌本身也在适应环境。

不知过了多久，脚下踩到了实地。

不是泥地，是碎石。沈言摸索着往前走了两步。空间像是开阔了些，头顶能感到空荡。他划亮火折子。

橙黄的光圈在黑暗里炸开，照亮三尺内的范围。

他站在一个天然石室的底部。石室不大，约莫三丈见方。四壁是粗糙的岩石，表面有细密的划痕。不像天然形成，更像是被什么东西反复刮擦过。地面碎石间积着浅浅的水，映着火光。

石室尽头，靠墙的位置，有一样东西。

沈言举高火折子。

那是一把剑。

剑身笔直，插在石缝里。没有剑鞘，剑刃蒙着厚厚的灰和锈迹。剑柄缠着的布条已经腐烂，露出底下暗沉的木头。

剑魂的嗡鸣在这时陡然清晰。

不是从脊椎传来，是从脑海深处。像有另一个人的心跳，重叠在他自己的心跳上。沉重，缓慢，带着铁锈般的涩意。

沈言握火折子的手指收紧了。竹筒发出轻微的咯吱声。

他没往前走。火光能照到的距离有限，剑周围的黑暗更浓。他注意到剑身插着的地方，石缝周围有深褐色的痕迹。不是锈，是血。年代久远的血迹渗进石头，洗不掉了。

“你来了。”

声音直接在脑海里响起。不是剑魂之前那种冰冷的震动，是真正的“声音”。苍老，疲惫，每个字都像磨过石头。

沈言的呼吸停了一瞬。

火光摇曳，剑身上的灰尘像是动了一下。不是风吹，是灰尘自己在褪去。露出底下暗银色的剑刃，刃口没有崩缺，只是蒙着一层类似锈迹的暗色。

“沈昊的儿子。”那声音又说。这次带了点审视的意味。“还是说……你就是沈昊？”

沈言没回答。他盯着那把剑。脊椎里的共鸣越来越强，差点要震得他发麻。那不是排斥，是辨认。剑魂在辨认同类。

“我不是沈昊。”沈言开口，声音在石室里有回音，显得比平时沉。“他是我父亲。”

“父亲……”声音重复了一遍这个词，像在咀嚼味道。“多久了？”

“三年前我被送到这里。”沈言顿了顿，“他失踪了。”

剑身上的锈迹褪得更快了。露出完整的剑刃，刃面光洁如水，映着火光。但剑柄依旧腐朽，缠布散落，木头上满是裂痕。

“失踪。”那声音笑了一下，很短促，没什么笑意。“他们也是这么对我说的。”

“他们是谁？”

“送你来的人。送这把剑来的人。”声音停顿，“封住我的人。”

沈言感到掌心发冷。不是因为石室阴湿。是火折子快灭了。竹筒里的火绒快烧尽了。

“你是剑魂？”他问。

“残魂。”声音纠正，“这把剑的主人死了，魂散了，只剩一缕执念留在剑里。久了，连执念也快磨没了。”

沈言往前走了一步。碎石在脚下嘎吱响。

“我父亲的剑魂也在我体内。”他说，“它认识你。”

石室里安静了。只有水滴从井壁落下的声音，叮，叮。

过了好一会儿，那声音才再次响起。

“沈昊……没死？”

“不知道。”沈言诚实地说，“剑魂认了我。但我没见过他。三年前的事，宗门没人告诉我真相。”

残魂沉默得更久了。火折子彻底灭了。黑暗合拢过来。沈言站在黑暗里，脊椎里的嗡鸣成了唯一的指引。

“真相……”残魂的声音低下去，“真相是柴饦峰不是善地。裂井也不是试炼场。是坟墓。”

沈言没动。黑暗里，他听见自己心跳的声音。一下，两下。

“明天辰时，我要下去参加逆魂试炼。”他说，“门主蒋阶让我来的。带着一块黑牌。”

残魂的嗡鸣忽然拔高，刺得沈言太阳穴一跳。那不是情绪，是某种频率的传递。

“黑牌……”残魂的声音压低了，“别让它碰水。尤其裂井底的水。牌子上的禁制会传声。”

沈言记住了。他问：“试炼到底是什么？”

“筛选。”残魂说，“能承受剑魂的容器，三年里，柴饦峰收了一百零三个弃徒。活到现在的，只有你一个。”

沈言喉咙发干。他舔了舔嘴唇，尝到泥腥味。

“其他人呢？”

“死了。或者废了。剑魂认主，不是请客吃饭。认不上的，就碎了。”残魂顿了顿，“你父亲……当年也是这么过来的。”

石室里的温度像是又降了些。沈言感到脊背发凉，不知是潮湿，还是别的。

“他为什么失踪？”

“因为他发现这宗门不对。”残魂的声音很轻，像快散了，“剑魂不是上古遗留，是人造的。柴饦峰是工坊，门主是匠人。你们这些弃徒，是材料。”

沈言的手指握紧了柴刀的木柄。木头粗糙的纹理硌着掌心。

“材料……做什么？”

“炼剑。”残魂说，“用活人的魂，炼剑魂。炼成了，剑魂养在人身上，人死了，剑魂抽出来，封进剑里。一把一把，攒着。”

井壁上的水滴声还在继续。叮，叮。

沈言没说话。他在黑暗里站了很久，直到眼睛能勉强看清剑的轮廓。暗银色的刃口在绝对的黑暗里，居然有一丝极淡的光。

“我父亲炼出来了？”

“炼出来了。极品。”残魂的声音带上点说不清的意味，“所以他得死。但剑魂不能废。得找个新的容器，把剑魂渡过去。渡不好的，容器就碎。渡得好，容器活，剑魂稳，就可以开始下一炉。”

沈言感到胃里一阵翻搅。他弯下腰，手撑着膝盖。喉咙里涌上酸水，他硬咽了回去。

“我就是那个新容器。”

“你是。”残魂说，“但你父亲的剑魂认了你，没碎。这很意外。上一个认主的，是一百二十年前的事了。”

沈言直起身。黑暗里，他的声音很稳。

“我明天还要下去。黑牌是监控，我不带，周奉会起疑。带进去，怎么避开？”

残魂沉默片刻。

“牌子最怕两样东西。水，和更上等的禁制。裂井底有积水，但不够深。更上等的禁制……”它顿了顿，“我这里没有。但你可以让它碰碰别的。”

“别的？”

“比如，我这把剑的剑柄。”残魂说，“上面的腐木不是真腐，是封禁。沾了黑牌，牌子上的禁制会紊乱一阵子。足够你走过前半段路。后半段……你自己小心。”

沈言点头。他摸出怀里用油纸包着的火折子。那是他从灶房顺的，没受潮。划亮，点燃一小截干草。火光重新亮起。

他走到剑前，蹲下身。

近距离看，剑身上的锈迹褪尽后，暗银色的刃口映着火光，流动着水一样的光泽。但剑柄确实朽烂了，木头裂成细条，缠布碎成絮状。

沈言把黑牌掏出来。牌面靠近剑柄的瞬间，脊椎里的麻意和剑身传来的嗡鸣同时剧烈起来。像两股水相遇，激起漩涡。黑牌内部的晦涩波动猛地一颤，然后乱了。像受惊的鱼群，四下冲撞。

有用。

沈言把黑牌在剑柄上贴了三息，然后收回怀里。牌面的麻意减弱大半，不再刺骨，只是沉闷地抵着皮肤。

“够了。”残魂的声音更飘忽了，“我撑不了多久。你该走了。”

沈言站起来。他看了剑最后一眼。

“你叫什么名字？”

“记不清了。”残魂说，“以前有人叫过我……沉渊。”

沈言记住了。他没再说话，转身走向井壁。攀爬上去时，他感到脊椎里的共鸣渐渐平息。剑魂安静下来，像睡着了。

爬出裂井时，天边已经泛白。

他站在井口，深吸一口凌晨清冷的空气。肺里那股霉味散了些。他低头看手，指尖的伤口凝了黑红色的血痂。

怀里的黑牌安静地贴着皮肉，不再震动。

沈言转身往柴饦峰走。靴子踩在湿泥上，声音比来时沉。

他得在辰时前赶回去，换身干净衣裳，把柴刀磨快。试炼要开始了。坟墓要开了。

他得进去，看看里头到底埋了多少骨头。
