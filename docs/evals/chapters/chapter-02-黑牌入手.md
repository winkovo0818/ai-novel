# 第 2 章《黑牌入手》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：2155

雨还没停。沈言站在大殿外的石阶下，粗麻短褐的下摆湿了半截，往下滴着水。带他来的杂役弟子早就走了，留下一句话：等着。

石阶很宽，青石板被雨水洗得发亮，映出昏沉沉的天光。沈言没往上走，就站在雨里。殿门紧闭，里面什么声音都听不见。他抬起手，抹了一把脸上的雨水，手指冰凉。指尖碰到额角时，他停顿了一下。那里有点发麻，不是冷的。是脊椎深处那点残余的震动还没散尽，像烧红的针尖扎过后留下的灼痛感。

剑魂。沈昊。

雨声太大，盖住了所有别的声音。他想起灶膛边那场短暂的意识接触。冰冷，直接，没有感情，只有陈述。等待。它认识沈昊，它在等他。三年前他被扔到柴饦峰，像一块没人要的破抹布。现在，抹布里面裹着一把也许能划破什么的旧刀。

他攥了攥拳，指甲陷进掌心。疼。这点疼让他清醒。不能想太多。至少不能在这里，在紧闭的殿门外，在不知道多少双眼睛底下。

脚步声从侧廊传来。

沈言没回头，脊背绷紧了半分。脚步声不疾不徐，水靴踩在湿石板上，发出特有的吧嗒声。他认得这声音。周奉走到他身边，也停在石阶下，和他并排站着，一同看着紧闭的殿门。

“门主在见内门几位长老。”周奉的声音被雨声冲得有些模糊，“关于小比的事。”

沈言低低应了一声：“嗯。”

“你三年考评，次次垫底。”周奉的目光落在他湿透的衣摆上，“按规矩，明天之后，你该卷铺盖走人了。”

沈言沉默。指甲又掐进掌心一点。规矩。柴饦峰的规矩就是废物没有第二条路。

“但门主给了你一个机会。”周奉从怀里掏出一样东西。那是一块巴掌大的木牌，黑沉沉的，边缘打磨得光滑。他把木牌递过来。

沈言接住。木牌入手很沉，不像是普通木头。指尖碰到牌面的瞬间，脊椎深处那点针扎似的灼痛感，猛地跳了一下。

不是错觉。剑魂在示警。

他手指蜷缩，险些把木牌捏碎。不能动。他强迫自己呼吸平缓，抬头看向周奉。周奉的脸上没什么表情，只有一双眼睛在雨幕后的昏暗光线下，显得格外深。

“这是‘试炼令’。”周奉说，“明天小比，所有未出师的弟子都要上台。但只有持此令者，才算正式参加‘逆魂试炼’。”

逆魂试炼。沈言听着这个词，心里沉了沉。光听名字就知道不是善茬。

“门主的意思是，你去参加这个试炼。”周奉盯着他，“如果能通过，不按垫底论处。不仅不逐出，还可直接列入内门候选。”

听起来像个天大的恩赐。沈言低头看手里的黑牌。牌面上刻着一个古篆的“试”字，笔画很深。指尖摩挲过去，能感觉到细微的凹凸。脊椎里的震动感又来了，这次更清晰，带着一股冰冷的、抗拒的意味。剑魂在排斥这块牌子。

为什么？

“拿着它，明天辰时，去后山裂谷入口集合。”周奉像是没注意到他细微的停顿，“别迟到。”

“裂谷……”沈言嗓子有点干。后山裂谷是禁地，听说下面镇着不干净的东西。

“试炼地点。”周奉打断他，语气没起伏，“怕了？”

沈言攥紧黑牌，指节发白。怕。当然怕。柴饦峰的人都知道后山裂谷意味着什么。但他没得选。不去，明天之后就是流落街头。去，至少还有一线机会。哪怕机会的另一头是悬崖。

“弟子遵命。”他听见自己干涩的声音。

周奉点点头，像是很满意他的顺从。他转身要走，又停住，像是随口提醒：“这牌子好生收着，别弄丢了。贴身带着。试炼要凭牌入场。”

贴身带着。沈言心里那根弦猛地绷紧。周奉的目光在他脸上一掠而过，平静无波。

“是。”沈言把黑牌揣进怀里，布料贴着牌面，那股冰冷的排斥感隔着衣服也丝丝缕缕地透进来，像揣着一块寒冰。他不得不挺直背，才没让颤抖显出来。

周奉这才真的走了。脚步声消失在侧廊尽头。

沈言一个人站在雨里。怀里的黑牌像块烙铁，也像块冰。剑魂的震动断断续续，每一次都伴随着细微的刺痛，顺着脊椎爬向四肢。它不喜欢这块牌子。它在警告他。

他抬起手，又按在额角。那里不再只是发麻。一种奇怪的感知顺着剑魂的震动延伸开来。他“看”不清，但能感觉到。怀里的黑牌深处，有一丝极细微、极晦涩的波动，像沉睡的毒蛇。不是木头本身的纹理，是后加上去的。像是……追踪？或者别的什么。

他想起刚才周奉的眼神，想起“贴身带着”那句话。

门主给的机会。周奉递来的牌子。三年垫底突然得到的试炼资格。剑魂的剧烈排斥。

这些碎片在雨声里拼凑起来，指向一个他不愿意深想的可能。

沈言站在原地，雨水顺着他额发淌下来，流进脖子里，冷得他打了个哆嗦。他没有掏出来看。不能看。看了，如果做出任何异常举动，周奉，或者别的什么人，可能就会知道。

知道他发现了。

他吐出一口气，白雾在冰冷的空气里迅速消散。好吧。去就去。试炼也好，陷阱也罢，总比被当成废物扫地出门强。至少，裂谷之下，或许还能找到别的线索。关于剑魂，关于沈昊，关于这三年浑噩背后到底藏着什么。

他最后看了一眼紧闭的殿门。门依然关着，里面的人，里面的谈话，像隔着一重天地。他转身，走下石阶，走进哗哗的雨幕里。怀里的黑牌贴着胸口，冰冷，沉重，像一颗埋进身体的种子。剑魂的震动还在，微弱地，固执地，透过皮肤，透过骨骼，提醒他一个事实：它醒了。而有些东西，比三年的冷眼和劈不完的柴更危险。

雨下得更大了。他裹紧湿透的衣襟，一步一步，踩着泥泞，朝柴饦峰的方向走去。黑牌在怀里，沉默地散发寒意。明天辰时，后山裂谷。

他得去弄明白，这牌子，和这试炼，到底想把他推向哪里。
