# 第 2 章《黑牌入手》

- 来源：llm
- 模型：mimo-v2.5-pro
- 字数：1504

柴饦峰的火房里还剩半扇门没掩上。孙奉没回头，蓑衣下摆扫过门框，带进一串湿泥脚印。他跨进两步，戒尺点在灶台边沿的泥点上，敲了敲。

“明日测骨，你不用去了。”

沈言垂着眼，看那戒尺黑亮的漆面上沾着灰。他喉咙动了动，没出声。灶膛里的柴又闷出一团烟，呛得他眯了下眼。

孙奉把戒尺往怀里一收，从袖中摸出一块木牌。巴掌大，边角磨得发圆，正面刻着“丙七”两个字，背面却光溜溜的，只一道新刻的痕。他往前一递。

“拿着。”

沈言抬头看他。孙奉的脸在昏暗里看不清神色，只有蓑衣滴水的声音，嗒，嗒，落在灶台边沿的灰泥里。

“这是什么？”

“明日考核的信物。”孙奉说，“拿着它，去后山裂井。找到井底那块断碑，把牌子按上去。做完了，算你过。”

沈言没动。他盯着那块木牌，指节收拢，抵在墙角的枯枝上。木头刺进皮肤，有点疼。他昨天才从裂井回来。井底有东西。那东西认得他父亲留下的剑魂，还发出过警告。

孙奉等了几息，戒尺抬起来，点在沈言肩膀上。不重，但硬。沈言肩胛骨往下一沉。

“不去？”孙奉的声音低下来，“不去也行。明日测骨，门主亲来。你这身根骨，瞒得过谁？”

沈言知道瞒不过。剑魂在体内已经醒了三个月，每天夜里，后背脊椎深处都会泛起一阵寒。像有冰顺着骨头缝往外钻。他咬牙忍着，白天在火房烧火，把柴灰往脸上抹，把衣服蹭得更脏。但孙奉的眼睛毒，蒋阶的眼睛更毒。他们等的就是测骨那一关。

“我去。”沈言说。声音有点哑。

孙奉戒尺往回一收。“现在就去。天亮前回来。牌子按上，碑会亮。不亮，你这丙七的名头，就没了。”

他把木牌往灶台上一放。牌子磕在石头沿上，响了一声。然后他转身往外走，蓑衣带起的风吹得灶膛里的火苗一歪。

沈言盯着那块牌子。

木头是泡过水的，颜色发深，摸起来有些潮。他拿起来，翻到背面。那道新刻的痕歪歪扭扭，像是匆忙间留下的。他用指甲刮了刮，痕很浅，但下面露出一小片不同的颜色。更深的黑，像是渗进木纹里的。

剑魂在体内动了一下。

不是寒气，是一种细微的嗡鸣，从后背升起来，顺着血管往手腕走。沈言手指一颤。他记得这感觉。昨天在裂井底，那道意识冲进他脑海时，也有过类似的嗡鸣。但这次更轻，更缓，像是某种低语。

他闭上眼睛。

火房里只剩下雨声和灶膛里柴火噼啪的响。嗡鸣在掌心停了停，然后往木牌里钻。沈言睁开眼，看见木牌背面那道刻痕下，渗出一点极淡的光。青灰色，像灶膛里将熄未熄的炭火。

光没亮起来，只是浮了一瞬。然后暗了。

沈言明白了。

这不是考核信物。这是追踪符。孙奉要他去裂井，但也要知道他到底会不会去，会不会做别的事。牌子按上碑，符可能就传了信号。不按，或者按晚了，孙奉那边也会知道。

他捏着牌子，指腹擦过那道痕。符没再亮。但木头底下，那层青灰色的痕迹还在，像一道暗疮。

他得去。

不去，孙奉现在就会折回来。去了，至少能拖到天亮。井底那道意识说过，剑魂认主不可逆。父亲留下的东西，不会害他。但孙奉要他死，蒋阶要他死。他不能死在明处。测骨那关，剑魂的寒气一放，就全完了。去裂井，是死路。不去，也是死路。

沈言把牌子揣进怀里。木头贴着胸口，隔着粗布，还是凉。他站起来，膝盖咔嗒响了一声。墙角枯枝的霉味钻进鼻子，混着雨潮气。他走到灶台边，把剩下的柴往里推了推，火苗旺了些，光映在墙上，影子一晃一晃。

然后他走到门边，没回头看。推开半扇门，雨水从屋檐滴下来，砸在他手背上，冷得刺骨。他跨出去，踩进泥里。泥没到脚踝，左脚鞋底那个洞灌进水，凉意顺着脚心往上爬。

夜很深。柴饦峰没有灯。只有雨。

沈言摸了摸怀里的牌子。那道刻痕硌在肋骨上。他往山后走。脚步声踩在水里，很快被雨声盖过去。
